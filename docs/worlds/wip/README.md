# Frostvale Valley — work-in-progress save

`frostvale-valley-wip.json` is the working save for the Frostvale Valley build
(brief: `docs/worlds/frostvale-valley-brief.md`). `.saves/` is gitignored and this
environment is ephemeral, so the WIP snapshot is committed here between phases as the
world's source material — it is not shipped content.

To resume building: copy it into `.saves/`, run `npm run dev`, and open
`?world=default&save=frostvale-valley-wip`.

Progress: Phase 0 (site survey) and Phase 1 (terrain + water) are complete — see the
brief's §5 phase list and §3 site record for details and gate results.
