# Ashen Reach — design statement (experimental)

**Status:** Grok candidate. Not official. Branch `grok/ashen-reach` only.

## World identity

| Field | Value |
| --- | --- |
| **Name** | Ashen Reach |
| **ID / preset** | `ashen-reach` |
| **Save identity** | `ashen-reach` (meta bootstrap in this folder) |
| **One-sentence** | A volcanic caldera kingdom where a glowing Ember Spire rises from a basalt island in a dark crater lake, watched by Emberhold village and a rim observatory. |
| **Atmosphere** | Warm dusk ash, black rock, molten glow, quiet settlement industry |
| **Architecture** | Brick/terracotta cottages, deepslate monuments, hip roofs, timber forges |
| **Terrain** | Authored radial caldera: flooded basin, ash terraces, basalt rim wall, outer slopes |
| **Primary landmark** | **The Ember Spire** — island tower with crystal/glowstone crown |
| **Secondary** | Emberhold plaza & gate, magma bridge, shore dock, vents, west observatory, east mine, ember shrine, south obelisk |

## Player journey

1. **Spawn** — Emberhold plaza, facing south toward the caldera.
2. **First reveal** — Spire beacon across the dark water; rim walls frame the bowl.
3. **Approach** — Ash road past forges and cottages down to the shore dock.
4. **Entry** — Magma fissure bridge on the east path (glow underfoot).
5. **Exploration** — Half-circuit of the lake: vents, mine adit, shoreline.
6. **Ascent** — Graded road + stairs climb the west rim to the observatory.
7. **Final viewpoint** — Observatory balcony / dome: whole caldera, Spire, village.

## Landscape-first notes

- Composition is a bowl, not a flat sandbox.
- Foreground: village bench. Mid: lake + spire. Background: basalt rim + sky.
- Road polyline grades terrain so the climb is walkable.
- No caves under architecture (solid ground).

## Non-goals

Survival, combat, scarcity. Does not replace Stonehaven, Frostvale, or shipped worlds.
